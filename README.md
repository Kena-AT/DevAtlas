# DevAtlas

**Your interactive map of modern software engineering knowledge.**

DevAtlas is a web-based educational reference app that centralizes modern software engineering concepts, best practices, tools, frameworks, architecture patterns, and commands across frontend, backend, database, integration, and deployment.

## Features

- **Domain Dashboards**: Dedicated views for Frontend, Backend, Database, Integration, and DevOps.
- **Interactive Guides**: Folder structures, code snippets, and architecture diagrams.
- **Search**: Global search across all concepts and frameworks.
- **Visual Learning**: Mermaid.js diagrams and syntax-highlighted code blocks.

## Tech Stack

- **Frontend**: React (Vite), Vanilla CSS (Modules/Variables), Framer Motion
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (optional/mock mode available)

## Getting Started

### Prerequisites

- Node.js (v18+)
- PostgreSQL (optional)

### Installation

1.  **Clone the repository**
2.  **Install Client Dependencies**
    ```bash
    cd client
    npm install
    ```
3.  **Install Server Dependencies**
    ```bash
    cd server
    npm install
    ```

### Running the App

1.  **Start the Backend**
    ```bash
    cd server
    npm run dev
    ```
2.  **Start the Frontend**
    ```bash
    cd client
    npm run dev
    ```
